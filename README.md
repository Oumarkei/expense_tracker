# 💰 Expense Tracker

Une application Streamlit pour suivre ses dépenses mensuelles, visualiser ses statistiques et gérer son budget.

## Fonctionnalités
- Ajouter, modifier et supprimer des dépenses
- Définir un budget mensuel
- Visualisations graphiques : par catégorie, par jour, cumulatif vs budget
- Export CSV des données

## Installation
```bash
git clone <repo_url>
cd ExpenseTracker
pip install -r requirements.txt
streamlit run app.py
